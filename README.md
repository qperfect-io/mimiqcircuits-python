# Circuits (`circuits-python`)

A library to handle quantum circuits for **QPerfect**'s MIMIQ Emulator.

## Installation
pip install circuits

## Usage

```python
import circuits
from circuits import gates, quantumcircuit
import numpy as np

circuit=quantumcircuit.Circuit()


```
